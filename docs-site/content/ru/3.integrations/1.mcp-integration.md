# Интеграция MCP

Через `@nuxtjs/mcp-toolkit` сайт документации экспортирует инструменты в формате MCP.

## Что включено

- Автопоиск инструментов из `server/mcp/tools/`
- Инструмент `docs-links` для локализованных URL
- Интеграция с AI-клиентами, поддерживающими MCP

## Как использовать

1. Запустите `npm run dev` в `docs-site`
2. Подключите MCP-клиент к endpoint приложения
3. Вызовите `docs-links` с нужной локалью

См. также: [Интеграция LLM](/ru/integrations/llm-integration).
