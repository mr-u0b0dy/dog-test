# Интеграция LLM

`nuxt-llms` добавляет машиночитаемые представления документации.

## Доступные маршруты

- `/llms.txt`
- `/llms-full.txt`

## Конфигурация

- Настройки секций и ссылок задаются в `nuxt.config.ts`
- Для production укажите `NUXT_PUBLIC_SITE_URL`

Это позволяет агентам быстрее находить релевантные страницы.
