# Написание тестов

Тесты в dog-test объявляются через семейство `DT_TEST` макросов.

## Минимальный пример

```c
DT_TEST("my_test", DT_RESET_NONE, NULL, my_test) {
  DT_ASSERT_TRUE(1);
  return DT_TEST_PASSED;
}
```

## Основные макросы

- `DT_TEST`
- `DT_TEST_T`
- `DT_TEST_F`
- `DT_TEST_FT`

В конце файла используйте `DT_MAIN()` для точки входа.
