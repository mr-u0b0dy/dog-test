# Monitor Specs

Monitor spec задаёт параметры захвата сигналов и ожидаемые условия верификации.

## Поля

- `protocol`: `uart`, `i2c`, `spi`
- `channels`: карта каналов
- `trigger`: условие старта
- `timeout_ms`: время захвата
- `expect`: условия проверки

## Пример

```json
{
  "protocol": "uart",
  "channels": "ch0=tx,ch1=rx",
  "trigger": "start-on-byte:0x55",
  "timeout_ms": 1200,
  "expect": { "protocol": "uart", "min_items": 1, "contains": "0x55" }
}
```

Некорректные ключи в `expect` вызывают ошибку валидации.
