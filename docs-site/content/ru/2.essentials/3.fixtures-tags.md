# Fixtures и теги

## Fixtures

`DT_TEST_F` позволяет выполнять инициализацию до теста и очистку после теста.

```c
static int setup(void) { return 0; }
static int teardown(void) { return 0; }
```

Если `setup()` возвращает не 0, тело теста не выполняется.

## Теги

Добавляйте теги через `DT_TEST_T` или `DT_TEST_FT`:

```c
DT_TEST_T("uart_smoke", DT_RESET_SOFT, NULL, "uart,smoke", uart_smoke) { ... }
```

Запуск по тегу: `--tag smoke`.
