# Dog Test Workflow

## Запуск одного теста

```bash [Terminal]
python3 tools/dt_runner.py \
  --firmware build/arm-gcc-debug/examples/uart_loopback/uart_loopback \
  --backend pyocd \
  --serial-port /dev/ttyACM0 \
  --reset-mode soft
```

## Запуск плана

```bash [Terminal]
python3 tools/dt_runner.py --test-plan tests/test_plan_smoke.json --junit-xml results.xml
```

## Полезные флаги

- `--retries`
- `--tag` / `--filter`
- `--monitor-spec`
- `--junit-xml`

Для CI запуск обычно оборачивается через CTest пресеты.
