# Python API

Python-часть расположена в каталоге `tools/`.

## CLI пример

```bash [Terminal]
dt-runner --firmware build/... --backend pyocd --serial-port /dev/ttyACM0
```

## Основные модули

- `tools/dt_runner.py`
- `tools/backends/flash/*`
- `tools/backends/reset/*`
- `tools/logic/decoders.py`
- `tools/specs/monitor_contract.py`

## Полезные опции

- `--test-plan`
- `--monitor-spec`
- `--retries`
- `--junit-xml`

Для деталей по классам и сигнатурам используйте исходный код Python модулей.
