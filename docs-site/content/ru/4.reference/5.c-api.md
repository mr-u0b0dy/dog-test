# C API Reference

Публичные заголовки находятся в `framework/include/dogtest/`.

## test_case.h

- `DT_TEST`, `DT_TEST_T`
- `DT_TEST_F`, `DT_TEST_FT`
- `DT_MAIN()` и функции запуска тестов

## assert.h

- `DT_ASSERT_*` — критичные проверки
- `DT_EXPECT_*` — накопление ошибок
- `DT_SKIP(reason)` — пропуск теста

## monitor_spec.h

- `DT_MONITOR_UART`
- `DT_MONITOR_I2C`
- `DT_MONITOR_SPI`

Полный перечень макросов смотрите в исходных заголовках.
