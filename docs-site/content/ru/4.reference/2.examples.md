# Примеры

В каталоге `examples/` подготовлены базовые сценарии для запуска и адаптации.

## Набор примеров

- UART loopback
- I2C smoke
- SPI transfer

## Быстрый запуск

```bash [Terminal]
python3 tools/dt_runner.py --test-plan tests/test_plan_smoke.json --backend pyocd
```

Каждый пример показывает типичный reset-mode и подход к проверке результатов.
