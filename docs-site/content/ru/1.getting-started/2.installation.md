# Установка

На этой странице собраны минимальные шаги для запуска dog-test локально.

## Необходимые инструменты

- CMake 3.22+
- Ninja
- GCC или Clang
- Python 3.10+

При кросс-сборке для ARM дополнительно нужен `arm-none-eabi-gcc`.

## Быстрый старт

```bash [Terminal]
git clone https://github.com/mr-u0b0dy/dog-test.git
cd dog-test
pip install -r requirements.txt
cmake --preset host-debug
cmake --build --preset host-debug
ctest --preset host-debug
```

## Запуск docs-site

```bash [Terminal]
cd docs-site
npm install
npm run dev
```

Проверка в браузере: `http://localhost:3000`.
