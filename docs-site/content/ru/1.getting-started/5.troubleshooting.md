# Устранение неполадок

## Ошибки flash/reset

- Проверьте подключение отладчика и питание
- Убедитесь, что `pyocd list` видит устройство
- На Linux проверьте права доступа (udev)

## Нет строк `DT_EVENT`

- Верный ли `--serial-port`
- Совпадает ли baudrate
- Проверены ли TX/RX/GND соединения

## Проблемы сборки

- Запустите `cmake --list-presets`
- Проверьте наличие нужного toolchain
- Убедитесь, что линковка включает `dogtest_core`

## Проблемы Python-зависимостей

```bash [Terminal]
pip install -r requirements.txt
```

Для Saleae при необходимости установите `saleae-automation`.
