---
title: Intégration LLM
---

Ce projet active une documentation adaptée aux LLM avec `nuxt-llms`.

## Ce qui est activé

- Route `/llms.txt` pour une doc concise lisible par machine
- Route `/llms-full.txt` pour un contexte étendu
- Métadonnées et sections configurées dans `nuxt.config.ts`

## Utilisation

1. Démarrez l'app docs avec `npm run dev`.
2. Ouvrez `/llms.txt` pour valider la sortie de base.
3. Ouvrez `/llms-full.txt` pour valider la sortie étendue.

## Production

Définissez `NUXT_PUBLIC_SITE_URL` avec l'URL publique du site pour générer des liens corrects.
