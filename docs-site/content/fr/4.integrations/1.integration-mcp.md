---
title: Intégration MCP
---

Ce projet active le support MCP avec `@nuxtjs/mcp-toolkit`.

## Ce qui est activé

- Métadonnées du serveur MCP dans `nuxt.config.ts`
- Découverte automatique des outils MCP depuis `server/mcp/tools`
- Outil d'exemple : `docs-links` pour les URLs d'intégration

## Utilisation

1. Démarrez l'app docs avec `npm run dev`.
2. Connectez un client MCP au point d'accès de l'application.
3. Appelez `docs-links` avec `locale: en` ou `locale: fr`.

## Notes

- Ajoutez d'autres outils dans `server/mcp/tools`.
- Gardez des réponses courtes et structurées pour les agents.
