# API Python

Les composants Python sont dans `tools/`.

## Exemple CLI

```bash [Terminal]
dt-runner --firmware build/... --backend pyocd --serial-port /dev/ttyACM0
```

## Modules principaux

- `tools/dt_runner.py`
- `tools/backends/flash/*`
- `tools/backends/reset/*`
- `tools/logic/decoders.py`
- `tools/specs/monitor_contract.py`

## Options utiles

- `--test-plan`
- `--monitor-spec`
- `--retries`
- `--junit-xml`

Pour le détail des signatures, se référer au code source Python.
