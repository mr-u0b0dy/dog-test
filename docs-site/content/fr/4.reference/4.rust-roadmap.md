# Feuille de route Rust

## Objectif

Exécuter des tests embarqués Rust via le même orchestrateur que C/C++.

## Contrat de compatibilité

Les binaires Rust doivent produire des lignes `DT_EVENT` compatibles.

## Étapes prévues

1. Adapter la build avec `cargo build --target`
2. Générer des plans de test depuis les métadonnées Rust
3. Ajouter des attributs reset/monitor
4. Supporter des suites mixtes C + Rust
