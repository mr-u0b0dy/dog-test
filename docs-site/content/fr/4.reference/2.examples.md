# Exemples

Le dossier `examples/` contient trois cas de démonstration.

- UART loopback
- I2C smoke
- SPI transfer

## Exécution rapide

```bash [Terminal]
python3 tools/dt_runner.py --test-plan tests/test_plan_smoke.json --backend pyocd
```

Chaque exemple illustre une combinaison de reset-mode et de vérification.
