# Référence API C

Les API publiques sont dans `framework/include/dogtest/`.

## `test_case.h`

- `DT_TEST`, `DT_TEST_T`
- `DT_TEST_F`, `DT_TEST_FT`
- `DT_MAIN()`

## `assert.h`

- `DT_ASSERT_*` (fatal)
- `DT_EXPECT_*` (non fatal)
- `DT_SKIP(reason)`

## `monitor_spec.h`

- `DT_MONITOR_UART`
- `DT_MONITOR_I2C`
- `DT_MONITOR_SPI`

Consultez les headers pour la liste complète des macros et types.
