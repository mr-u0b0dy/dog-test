# Dépannage

## Flash / reset

- Vérifier la sonde de debug et l’alimentation
- Contrôler la détection via `pyocd list`
- Vérifier les permissions USB (udev sous Linux)

## Pas de lignes `DT_EVENT`

- Port série correct (`/dev/ttyACM0`, etc.)
- Baudrate cohérent avec le firmware
- Câblage TX/RX/GND correct

## Erreurs de build

- `cmake --list-presets`
- Toolchain ARM disponible si nécessaire
- Lien avec `dogtest_core`

## Dépendances Python

```bash [Terminal]
pip install -r requirements.txt
```
