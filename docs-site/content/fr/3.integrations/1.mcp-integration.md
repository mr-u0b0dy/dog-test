# Intégration MCP

Le module `@nuxtjs/mcp-toolkit` permet d’exposer des outils MCP pour les assistants IA.

## Ce qui est activé

- Découverte automatique des outils dans `server/mcp/tools/`
- Outil `docs-links` pour récupérer les URLs localisées
- Utilisation par des clients compatibles MCP

## Utilisation

1. Lancer `npm run dev` dans `docs-site`
2. Connecter le client MCP au serveur
3. Appeler `docs-links` avec la locale désirée

Voir aussi : [Intégration LLM](/fr/integrations/llm-integration).
