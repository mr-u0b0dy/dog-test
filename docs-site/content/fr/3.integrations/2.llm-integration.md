# Intégration LLM

`nuxt-llms` publie des routes textuelles optimisées pour les LLM.

## Endpoints

- `/llms.txt`
- `/llms-full.txt`

## Configuration

- Définir les sections et liens dans `nuxt.config.ts`
- En production, renseigner `NUXT_PUBLIC_SITE_URL`

Les agents peuvent ainsi explorer la documentation de façon fiable.
