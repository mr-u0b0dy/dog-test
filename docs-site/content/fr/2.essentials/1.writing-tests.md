# Écrire des tests

Les tests sont déclarés via les macros `DT_TEST`.

## Forme de base

```c
DT_TEST("my_test", DT_RESET_NONE, NULL, my_test) {
  DT_ASSERT_TRUE(1);
  return DT_TEST_PASSED;
}
```

## Macros principales

- `DT_TEST`
- `DT_TEST_T`
- `DT_TEST_F`
- `DT_TEST_FT`

Ajoutez `DT_MAIN()` en fin de fichier pour générer le point d’entrée.
