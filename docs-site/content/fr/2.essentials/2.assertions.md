# Assertions

dog-test propose deux familles d’assertions :

- `DT_ASSERT_*` : échec fatal, arrêt immédiat du test
- `DT_EXPECT_*` : échec enregistré, exécution continue

## Exemples courants

- `DT_ASSERT_EQ(expected, actual)`
- `DT_ASSERT_TRUE(expr)`
- `DT_ASSERT_STR_EQ(a, b)`
- `DT_EXPECT_EQ(expected, actual)`

Utilisez `DT_EXPECT_*` pour collecter plusieurs erreurs dans un même test.
