---
title: MCP Integration
---

This project enables MCP support with `@nuxtjs/mcp-toolkit`.

## What is enabled

- MCP server metadata in `nuxt.config.ts`
- Auto-discovered MCP tools from `server/mcp/tools`
- Example tool: `docs-links` for integration URLs

## Use it

1. Start the docs app with `npm run dev`.
2. Connect an MCP client to the app endpoint.
3. Call `docs-links` with `locale: en` or `locale: fr`.

## Notes

- Add more tools under `server/mcp/tools`.
- Keep tool responses concise and structured for agents.
