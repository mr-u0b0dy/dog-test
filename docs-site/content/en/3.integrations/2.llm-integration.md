---
title: LLM Integration
---

This project enables LLM-friendly documentation with `nuxt-llms`.

## What is enabled

- `/llms.txt` route for concise machine-readable docs
- `/llms-full.txt` route for expanded docs context
- Site metadata and sections configured in `nuxt.config.ts`

## Use it

1. Start the docs app with `npm run dev`.
2. Open `/llms.txt` to verify base output.
3. Open `/llms-full.txt` to verify extended output.

## Production setup

Set `NUXT_PUBLIC_SITE_URL` to your public site URL so generated links use the correct domain.
