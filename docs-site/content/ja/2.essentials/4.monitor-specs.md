# Monitor Spec

monitor spec は、信号捕捉の条件と期待値を JSON で定義する仕組みです。

## 代表フィールド

- `protocol`: `uart` / `i2c` / `spi`
- `channels`: チャネル割り当て
- `trigger`: 捕捉開始条件
- `timeout_ms`: タイムアウト
- `expect`: 期待値ルール

## 例

```json
{
  "protocol": "uart",
  "channels": "ch0=tx,ch1=rx",
  "trigger": "start-on-byte:0x55",
  "timeout_ms": 1200,
  "expect": { "protocol": "uart", "min_items": 1, "contains": "0x55" }
}
```

`expect` の不正キーはエラーになります。
