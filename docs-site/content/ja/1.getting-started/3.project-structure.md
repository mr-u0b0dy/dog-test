# プロジェクト構成

dog-test は C フレームワークと Python ランナーで構成されています。

## 主要ディレクトリ

- `framework/`: 組み込み側テスト実行とアサーション
- `tools/`: `dt_runner.py` とバックエンド実装
- `examples/`: UART / I2C / SPI のサンプル
- `tests/`: CTest と Python テスト
- `docs-site/`: このドキュメントサイト

## 開発の流れ

1. CMake でビルドを構成
2. ファームウェアをビルド
3. `dt_runner.py` で実行・評価
4. CTest で自動化

詳細は [Dog Test Workflow](/ja/reference/dt-workflow) を参照してください。
