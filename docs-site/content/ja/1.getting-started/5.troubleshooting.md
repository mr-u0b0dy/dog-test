# トラブルシューティング

## フラッシュやリセットに失敗する

- デバッグプローブの接続と電源を確認
- `pyocd list` で認識状態を確認
- Linux の場合は udev 権限を確認

## シリアルに `DT_EVENT` が出ない

- ポート名（例: `/dev/ttyACM0`）が正しいか確認
- ボーレートをファームウェア設定と一致させる
- 配線（TX/RX/GND）を見直す

## ビルドエラーが出る

- プロジェクトルートで `cmake --list-presets` を実行
- `arm-none-eabi-gcc` が必要なプリセットか確認
- `target_link_libraries(... dogtest_core)` を確認

## Python 依存で失敗する

```bash [Terminal]
pip install -r requirements.txt
```

必要に応じて `saleae-automation` を追加してください。
