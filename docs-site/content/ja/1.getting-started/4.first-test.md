# 最初のテスト

ここでは、最小の C テストを作成して実行するまでを説明します。

## 1. テストコードを作る

```c
#include "dogtest/test_case.h"
#include "dogtest/assert.h"

DT_TEST("addition_works", DT_RESET_NONE, NULL, addition_works) {
  DT_ASSERT_EQ(2 + 2, 4);
  return DT_TEST_PASSED;
}

DT_MAIN()
```

## 2. ビルド設定

`dogtest_core` をリンクした実行ファイルを作成してください。

## 3. 実行

```bash [Terminal]
cmake --preset host-debug
cmake --build --preset host-debug
ctest --preset host-debug
```

次は [Assertions](/ja/essentials/assertions) で失敗時の挙動を確認します。
