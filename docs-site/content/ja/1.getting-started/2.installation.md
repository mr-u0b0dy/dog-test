# インストール

このページでは、dog-test をローカル環境で実行するための最短手順を説明します。

## 必要なツール

- CMake 3.22 以上
- Ninja
- GCC または Clang
- Python 3.10 以上

## 基本セットアップ

```bash [Terminal]
git clone https://github.com/mr-u0b0dy/dog-test.git
cd dog-test
pip install -r requirements.txt
cmake --preset host-debug
cmake --build --preset host-debug
ctest --preset host-debug
```

## docs-site の起動

```bash [Terminal]
cd docs-site
npm install
npm run dev
```

ブラウザで `http://localhost:3000` を開いて表示を確認します。
