# LLM 連携

`nuxt-llms` により、LLM が参照しやすいテキストエンドポイントを提供します。

## 提供エンドポイント

- `/llms.txt`: 要約版
- `/llms-full.txt`: 詳細版

## 設定ポイント

- `nuxt.config.ts` の `llms` セクションでタイトルやリンクを定義
- 本番では `NUXT_PUBLIC_SITE_URL` を正しく設定

AI クライアントはこの情報からドキュメント構造を把握できます。
