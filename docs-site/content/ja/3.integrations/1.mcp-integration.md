# MCP 連携

docs-site では `@nuxtjs/mcp-toolkit` により MCP クライアント向けツールを提供できます。

## 有効化ポイント

- `nuxt.config.ts` で MCP モジュールを有効化
- `server/mcp/tools/` 配下のツールを自動検出
- AI エージェントが `docs-links` などを呼び出し可能

## 運用の流れ

1. `npm run dev` で docs-site を起動
2. MCP クライアントからエンドポイントへ接続
3. `docs-links` で言語別 URL を取得
