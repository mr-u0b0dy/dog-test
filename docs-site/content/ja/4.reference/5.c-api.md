# C API リファレンス

公開 API は主に `framework/include/dogtest/` の 3 ヘッダーです。

## `test_case.h`

- `DT_TEST`, `DT_TEST_T`, `DT_TEST_F`, `DT_TEST_FT`
- `DT_MAIN()`
- `dt_run_all_tests()` などの実行関数

## `assert.h`

- 致命系: `DT_ASSERT_*`
- 非致命系: `DT_EXPECT_*`
- スキップ: `DT_SKIP(reason)`

## `monitor_spec.h`

- `DT_MONITOR_UART`
- `DT_MONITOR_I2C`
- `DT_MONITOR_SPI`

実際の定義と全マクロはヘッダーファイルを参照してください。
