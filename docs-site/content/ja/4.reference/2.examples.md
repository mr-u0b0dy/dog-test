# サンプル

`examples/` には 3 つの代表ケースが含まれます。

## UART Loopback

送受信値が一致するかを確認する最小ケースです。

## I2C Smoke

I2C アドレス応答を確認するスモークテストです。

## SPI Transfer

転送後 CRC を検証するケースです。

## 実行例

```bash [Terminal]
python3 tools/dt_runner.py --test-plan tests/test_plan_smoke.json --backend pyocd
```

詳細は [Dog Test Workflow](/ja/reference/dt-workflow) を参照してください。
