# Python API

ホスト側 API は `tools/` 配下で提供されます。

## CLI

```bash [Terminal]
dt-runner --firmware build/... --backend pyocd --serial-port /dev/ttyACM0
```

## 主なコンポーネント

- `tools/dt_runner.py`: 実行オーケストレータ
- `tools/backends/flash/*`: フラッシュ実装
- `tools/backends/reset/*`: リセット実装
- `tools/logic/decoders.py`: UART/I2C/SPI デコード
- `tools/specs/monitor_contract.py`: monitor spec 検証

## 代表オプション

- `--test-plan`
- `--monitor-spec`
- `--retries`
- `--junit-xml`

詳細な関数シグネチャは Python ソースを参照してください。
